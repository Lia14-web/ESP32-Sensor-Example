
# ESP32 Sensor Example

Simple Arduino code for ESP32 to simulate a sensor reading. Demonstrates basic embedded programming structure.

## Hardware Required
- ESP32 development board

## How to Use
1. Open the `.ino` file in Arduino IDE
2. Select your ESP32 board (Tools > Board > ESP32 Dev Module)
3. Upload the code
4. Open Serial Monitor at 115200 baud

## What This Shows
- Serial communication setup
- Analog reading (`analogRead`)
- Non-blocking timing (using `millis()`)
- Simple LED blinking as system indicator
